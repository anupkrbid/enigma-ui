Form Elements

Forms can be composed of input, select, textarea elements, and other specialized input elements, such as checkboxes, and radio buttons.

Inputs

Below is an examples that shows how inpus can show validation state. Create a container with a class of .form-group which will hold indevidual form elements. .form-group is used to hold the main input and other addons. Adding .required along with the .label class shows an asterisk which indecated requied field. .input-group-prepend and .input-group-append is used to add addons on the left and right side on the input. .form-help-text is used to show the error messages or any info under the input. adding a class of .form-control-invalid will show the invalid state of the form.
