Navigation

Documentation and examples for Finess UI's powerful, responsive navigation header, the navbar. Includes support for branding, navigation, and more, including support for our dropdown component.

Simple Responsive Navigation

Below are examples of a simple responsive navigation using the .nav class. This is just a wrapper which lets you add brand name using the .brand-name class which should be inside a .nav-brand-container. And you can add links in the navigation using .nav-link and a .nav-content as sibling to show rest of the navigation links and .nav-link-active to highlight the active link. The internal responsivenes is on the dev to implmented as demonstrated.
