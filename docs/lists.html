Lists

List are a flexible and powerful component for displaying a series of content. Modify and extend them to support just about any content within.

Vertical Lists

Below are examples of a vertical list. Add a .list class to a ol or ul to create a liat. You can add anything you want in the container its your choice. .list dosn't have any width of it's own. it will take the width of the container it is in. You can add .list-items-spaced to add some padding around the items. You can add .list-items-gapped for additional space between the items.
