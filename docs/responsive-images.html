Rounded Images

Below are examples of rounded images. They can clip an image of any size to create a rounded image. Add a .image-responsive-container to a container and add an img inside it. You can add .image-responsive-container-rounded to create a rounded corner. You can also add .image-responsive-container-large to increase the size.
