Modal

Modals are dialogs which can be used for forms, notifications, lightboxes, and other custom content. This requires the enigma.js to be imported.

Simple Modal

This modal should be a direct child of the body. Create a container with .modal-overlay and a .modal-container inside it which will have 3 containers .modal-header for the heading of the modal , .modal-body for any content the user wants to put which is totally customizable and .modal-actions which will have the buttons to interact with the modal.
